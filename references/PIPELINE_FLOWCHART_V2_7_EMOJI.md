🔄 Spawn 重试机制

┌─────────────────────────────────────────────────────────────────────────┐
│  任何 agent spawn 失败时：                                               │
│  1️⃣ 第一次失败 → 立即重试（相同参数）                                    │
│  2️⃣ 第二次失败 → 等待 10 秒后重试                                        │
│  3️⃣ 第三次仍失败 → 推送告警到监控群 + 通知晨星 → 标记 BLOCKED            │
└─────────────────────────────────────────────────────────────────────────┘
